Build Tools & Typescript

by Emma Ramirez 2016-06-06 04:47:22pm

Typescript has, seemingly overnight, blown up onto the development scene. With over one million downloads monthly over npm, Typescript is now the most popular compile-to-javascript language. It makes sense, too, after all, Typescript bears a lot of features that make Javascript more stable—and in my opinion, more fun—to write. This article covers different builds in Typescript using Grunt, Gulp, Webpack, and SystemJS. This tutorial assumes you've played around with Typescript before and you understand some of ES6, but you can probably get by even if you don't. It also assumes you have node and npm installed.
  • Vanilla
  • Grunt
  • Gulp
  • Webpack
  • SystemJS

Downloading Typescript

Getting Typescript, if you don't already, is pretty easy. Just runnpm install typescript -gor, you're itching for the next release,npm install typescript@latest -gThe -g ensures that it is installed globally, but you will most likely want it locally in your projects as well. We'll cover that later.
